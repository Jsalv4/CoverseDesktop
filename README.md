# Coverse Standalone

Desktop app for Coverse collaboration workflows.

## What this repository contains

- Electron app source code
- Packaging/build configuration
- Update-capable desktop client

## Local development

```powershell
npm install
npm start
```

## Build (Windows)

```powershell
npm run build:win
```

## Releases

Installers and update metadata are published in the repository **Releases** section.

## Security

- Do not commit secrets or credentials.
- Keep environment-specific values in local environment files or CI secrets.

## Notes

Additional operational runbooks and implementation notes are maintained in project docs files.
